This is a simple line-following robot built using Raspberry Pi and Python. The robot reads data from two sensors (left and right) and decides how to move based on the line position. 
If both sensors detect the line, it moves forward. If only one sensor detects the line, it turns in that direction. If no line is detected, the robot stops. Motor speed is controlled using PWM.
This project helped me understand basic robotics logic, sensor reading, and motor control. In the future, I plan to improve it by adding PID control and better optimization.
